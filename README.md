# 🎓 Education Website

A modern and responsive education website built using HTML, CSS, and JavaScript. The website provides an attractive interface for students to explore courses, register, and learn through an interactive experience.

## ✨ Features

- Responsive design
- Modern UI/UX
- Course section
- Student registration page
- Interactive navigation
- Mobile-friendly layout
- Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Education Website/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md
```

## 🚀 Getting Started

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
